# voice-chat-gemini

## 專案簡介
多模態（語音、文字、臉部）情緒辨識與聊天機器人，支援 Gemini API、Flask API、語音互動。

## 主要功能
- 語音、文字、臉部情緒辨識
- 多模態情緒融合與紀錄
- Gemini API 智慧聊天
- 聊天與情緒歷史紀錄
- API 支援前端/Android

## 安裝方式
```bash
pip install -r requirements.txt
```
